AI app docs

Documentation

Multi-provider AI app — reports, chat, dual chat, RAG knowledge bases, on-device LLM & embedder, share-target ingest, fan-out / fan-in. 42 cloud providers across three API formats, plus a synthetic on-device Local provider.

Kotlin 2.2.10 · Jetpack Compose · single Activity · MVVM with five view models · seven external metadata repositories layered into one resolved view per (provider, model) pair.

For end users

Manual Walkthrough of every screen and feature — Reports, Chat, Dual Chat, Knowledge, Local, Translation, Fan-out, Housekeeping.
